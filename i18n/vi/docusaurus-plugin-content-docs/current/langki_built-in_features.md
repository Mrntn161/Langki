# Các tính năng tích hợp sẵn trong Langki

Mẫu thẻ Langki đi kèm với các tính năng tích hợp sẵn như chuyển văn bản thành giọng nói (text-to-speech), chuyển giọng nói thành văn bản (speech-to-text), và phân tích giọng nói (speech analysis).

Để chuyển văn bản thành giọng nói, sử dụng cú pháp `((tts:text))` hoặc `((tts_autoplay:text))`. Phiên bản `autoplay` sẽ tự động phát âm thanh khi thẻ được hiển thị. Ngoài ra, `((speech))` và `((speech:script))` được dùng khi bạn muốn tích hợp tính năng phân tích giọng nói vào flashcard.
